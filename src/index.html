How to install it?

Follow the next steps to install the Free PBX Server.

  • • Open Virtual Box.
  • • Select the New option, to create a new virtual machine.
  • • Fill the name input (eg.VOIPSERVER).
  • • Select the folder, where you want to create the new virtual machine.
  • • Select the ISO file of the server.
  • • Click the Finish button.
  • • Go to settings.
  • • Select the option Network.
  • • Change the Network Adapter from NAT to Bridged Adapter.
  • • Start the Virtual Machine that you just created.

    • This will show up, here you choose one version of Free PBX.

    I will be installing the third option.

  • Then this screen will apear and you will choose one output option.

    • After that, this will show up in your screen,

    just click enter.

  • •Wait until this show up
  • If you want to add a password just click in the ROOT PASSWORD option and set the password
  • • Wait for the end of the instalation process then click Reboot
  • • Close your virtual machine.
  • • Open virtual box.
  • • Go to settings.
  • • Go to storage.
  • • Then remove the instalation midia.
  • • Start your machine again.

    • When this apears

    Input root for username, and for the password input the one that you just set on the earlier stage. If you didn't set any, just click enter.

    This will show up and your server is on!!

  • Access the graphical user interface of Free PBX by searching the provided IP Adress in the browser.

    • In the first interaction there will be a sign up page just fill it, then click Setup System.

    This will show up

  • • Select the Free PBX Administration option.
  • • Choose the language.

    • And finaly when this show up

    Just click the Abort option then Apply Config.
